## AI 基础 — 上下文（Context）（70 分钟）

### 本章核心一句话

> Context 是模型能「看到」的一切——它的**上限**决定了推理的边界，它的**内容**决定了推理的质量。

### 学习重点

**1. 上下文是什么？（10 分钟）**
- 模型在生成每一个 token 时，能参考的全部信息：系统 Prompt、对话历史、工具返回值、检索文档
- 模型没有对话外的「记忆」，窗口之外的信息对它来说不存在
- 窗口内的所有内容都会影响输出，顺序和位置也有影响

**2. 四个关键概念（20 分钟）**

| 概念 | 一句话理解 |
|------|-----------|
| Context Window | 模型一次能处理的最大 token 数（如 Claude 支持 200K tokens） |
| Token | 模型处理文本的基本单位，约 0.75 个英文单词 / 1.5 个中文字 |
| Context Poisoning | 攻击者将恶意内容注入上下文，劫持模型行为，在 Agent 场景中危害极大 |
| Lost in the Middle | 上下文过长时，模型倾向于忽略中间部分，关键信息应放在首尾 |

**3. 三种上下文管理策略（20 分钟）**

| 策略 | 做法 | 适用场景 |
|------|------|---------|
| 摘要压缩 | 将旧对话轮次压缩为摘要，释放 token 空间 | 长对话 |
| 检索增强（RAG） | 只把与当前问题相关的文档块加入上下文 | 知识库问答 |
| 信任分层 | 系统 Prompt > 用户输入 > 工具输出 > 外部文档 | Agent 安全 |

**4. 在 AI × Web3 里的位置（20 分钟）**
- 上下文可以携带钱包地址、合约 ABI、链上交易历史——Agent 的「工作台」
- Context Poisoning 在链上场景中尤其危险：伪造的上下文可能导致 Agent 签署恶意交易
- 防御原则：**高风险参数（金额、地址、合约函数）必须在代码层校验**，不能只依赖上下文内容

完整链路回顾：
```
Prompt（定义任务）→ Context（注入数据）→ Model（生成结果）
→ Code（校验参数）→ Guard/Simulation（检查链上影响）→ Human Check（确认高风险）
```

### 我的笔记（学完后填写）

**用自己的话解释：为什么「上下文窗口有限」会影响 Agent 的可靠性？**
上下文窗口不宜过多也不能过少，过多会增加信息权重的分析，产生错误的偏侧和重点，容易让agent抓不住重点，过少则分析的信息又太少，无法准确高效完成任务

**Context Poisoning 和 Prompt Injection 有什么联系和区别？**
本质上都是导致模型输出不准确甚至错误的信息，区别在于prompt injection是一种有危害的骇客攻击，context poisoning则只是影响输出的答案的准确性，危害没有那么大

有哪些典型操作会导致context poisoning？

## Web3 基础 — 智能合约（Smart Contract）（20 分钟）

### 本章核心一句话

> 智能合约是部署在链上的代码，条件满足时**自动执行**——不依赖中间人，也几乎无法撤回。

### 学习重点（轻量阅读，建立直觉）

**五个基础概念：**

| 概念 | 一句话理解 |
|------|-----------|
| 智能合约 | 链上可执行代码，被调用时自动运行，无需人工干预 |
| ABI | 合约接口规范，告诉外部调用者有哪些函数、参数和返回值 |
| 部署（Deploy） | 将合约字节码发布到链上，一次性操作，消耗 Gas，地址永久固定 |
| 不可变性（Immutability） | 部署后代码无法修改，Bug 也会永久存在（除非使用升级模式） |
| 事件（Event） | 合约执行时写入链上的日志，链下应用可监听和查询 |

**三类常见安全风险：**

| 风险 | 一句话理解 | 风险级别 |
|------|-----------|---------|
| 重入攻击（Reentrancy） | 合约在更新状态前被外部合约回调，导致资金被反复提取 | 高 |
| 权限控制缺失 | 关键函数未加访问限制，任何人都能调用 | 高 |
| 整数溢出（Overflow） | 算术溢出导致金额计算错误（Solidity 0.8+ 已内置检查） | 中 |

### 我的笔记（学完后填写）

**用自己的话解释：智能合约的「不可变性」对开发者意味着什么？**
意味着开发者需要更加注意自己写的代码和函数，不能出现重大错误或者漏洞，否则一旦上链会导致资金损失等严重后果
