# Handbook Feedback

> 记录学习过程中发现的 Handbook 问题、卡点、建议。

## 使用方式

每次遇到以下情况，在本目录新建一个 `.md` 文件：

- 概念解释不清楚
- 资料过期或链接失效
- 错别字或排版问题
- 结构上的建议

## 文件命名

```
YYYY-MM-DD-[简短描述].md
```

例：`2026-05-18-module1-link-broken.md`

## 每条 Feedback 包含

| 字段 | 说明 |
|------|------|
| Handbook 页面链接 | 出问题的具体页面 |
| 问题描述 | 清楚说明发现了什么 |
| 建议改法 | 你认为怎么改会更好 |
| 来源 | 自己发现 / 讨论中提到 / 其他 |
