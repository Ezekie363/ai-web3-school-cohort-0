# 个人学习计划

> 学员：Ezekie363 | 更新：2026-05-18

## 学习节奏

- 每日投入：1–2 小时
- 核心方向：Hackathon 项目 + 产品/研究
- 工具策略：无代码背景，优先用 AI 工具（Claude/ChatGPT）辅助理解和操作

## 三条学习路径（按每日选择）

### 最小路径（30–45 分钟）
> 适合时间紧或状态一般的日子

1. 阅读 Handbook 当日章节（15 分钟）
2. 用自己的话写 1 条核心概念摘要
3. 更新 `daily/YYYY-MM-DD.md`

### 推荐路径（1–1.5 小时）
> 日常节奏

1. 读 Handbook 章节 + 相关资料（20 分钟）
2. 完成 WCB 平台当日任务（30 分钟）
3. 记录笔记和 1 个问题或卡点
4. 更新 daily note，生成打卡草稿

### 挑战路径（1.5–2 小时）
> 状态好或有课程项目时

1. 推荐路径全部完成
2. 在 `experiments/` 或 `hackathon/` 下动手探索一个想法
3. 写 Handbook feedback（如有卡点或建议）

## Hackathon 准备阶段目标

- [ ] 完成 Handbook 基础模块学习
- [ ] 积累 10+ 个 daily notes
- [ ] 在 `hackathon/` 下记录至少 3 个项目想法
- [ ] 参与至少 1 次黑客松或展示
